A small program that transmits sound from one user's microphone and broadcasts it to another (others)
===========

<details>
<summary>Addictions</summary>

How To (Install)
------
*  Deploy `server/server.py` somewhere on your machine so that the client can connect to it.
*  Download Client files in `udp-chat` and customize them to suit your needs.
*  Enjoy!
